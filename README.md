2. Identify and construct appropriate prompts for each task using an LLM (e.g., ChatGPT).
3. Simulate natural user interaction through a simple interface or command-line system.

Personal Productivity Assistant Features:

# Reflection Note

The experiment demonstrated that prompt refinement significantly improves the quality and usefulness of AI-generated task management solutions. Simple prompts generated generic responses, while advanced prompts produced more organized, personalized, and practical outputs.

Role-based prompts improved personalization, and structured prompts enhanced clarity and task organization.

The prompts could be refined further by:

- Including priority scoring
- Requesting visual charts
- Adding reminders and notifications
- Integrating productivity analytics
